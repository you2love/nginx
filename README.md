# Nginx 深度学习 - 内部机制与配置详解

一个现代化的Nginx深度学习教程，重点关注内部机制、请求处理流程、指令详解和应用配置，通过大量图表和流程图帮助理解。

## 项目结构

```
nginx-tutorial/
├── index.html           # 首页 - 学习路径导航
├── mechanism.html       # 内部机制 - 架构、模块、事件驱动
├── request-flow.html    # 请求处理流程 - 流程图、序列图
├── directives.html      # 指令详解 - 分类、表格展示
├── configs.html         # 应用配置 - 反向代理、负载均衡、缓存等
├── styles.css           # 样式文件
└── script.js            # 交互脚本和图表渲染
```

## 页面说明

### 1. index.html - 首页
- 网站导航入口
- 学习路径概览
- 核心概念预览

### 2. mechanism.html - 内部机制
- Master-Worker架构图
- 模块系统说明
- 事件驱动模型
- 内存池机制

### 3. request-flow.html - 请求处理流程
- 11个处理阶段流程图
- 请求生命周期序列图
- 各阶段交互图
- 核心函数调用表

### 4. directives.html - 指令详解
- 核心指令分类表
- 各模块指令详解
- 上下文与作用域
- 最佳实践建议

### 5. configs.html - 应用配置
- 反向代理配置
- 负载均衡策略
- 缓存配置详解
- SSL/TLS配置
- 安全配置

## 技术栈

- 纯HTML5 + CSS3
- SVG图表和流程图
- JavaScript交互
- 响应式设计

## 本地运行

直接在浏览器中打开 `index.html` 文件即可查看。
