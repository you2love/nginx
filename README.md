# Nginx 教程 - AI时代的高性能Web服务器

一个现代化的Nginx学习教程，包含指令详解、AI工具介绍和实战案例。

## 项目结构

```
nginx-tutorial/
├── index.html        # 主页（导航页）
├── commands.html     # 指令与配置
├── ai-tools.html     # AI工具与Nginx
├── practice.html     # 实战篇 - 最佳实践
├── resources.html    # 学习资源
├── styles.css        # 全局样式
└── script.js         # JavaScript脚本（保留，暂未使用）
```

## 使用方法

直接在浏览器中打开 `index.html` 即可开始学习。

所有页面都可以独立访问，通过导航栏在各个页面之间切换。
