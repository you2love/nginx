# Nginx 深度学习 - 内部机制与配置详解

一个现代化的Nginx深度学习教程，重点关注内部机制、请求处理流程、指令详解和应用配置，通过大量图表和流程图帮助理解。

## 项目结构

\`\`\`
nginx-tutorial/
├── index.html           # 首页 - 学习路径导航
├── mechanism.html       # 内部机制 - 架构、模块、事件驱动
├── request-flow.html    # 请求处理流程 - 流程图、序列图
├── directives.html      # 指令详解 - 分类、表格展示
├── configs.html         # 应用配置 - 反向代理、负载均衡、缓存等
├── styles.css           # 样式文件
└── script.js            # 交互脚本和图表渲染
\`\`\`
